# Application Shell

## Overview

A modern, luxurious application shell featuring a floating "island" top navigation bar. The design prioritizes content visibility with a full-screen hero aesthetic in mind.

## Navigation Structure

**Centered Links (The Island):**

- The Library (Browse Collections)
- Book Companion (Featured/Spotlight)
- About Us (Our Values)

## User Menu (Top Right)

- **Wishlist:** Icon button for saved items.
- **The Counter (Cart):** Icon button with badge.
- **User Avatar:** Dropdown menu for profile and logout.

## Design Decisions

- **Floating Island:** Creates a distinct, premium feel.
- **Glassmorphism:** Adds depth and modern texture.
- **Playfair Display:** Used for navigation links to match the brand.

## Components Provided

- `AppShell` — Main layout wrapper
- `MainNav` — The floating island navigation
- `UserMenu` — Top right actions
